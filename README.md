# arduino_sigfox_weather_dust
Weather and Dust Values send to Sigfox Backend ina n 15 Minute Intervall.

# Parts needed
## Board
Arduino MKR Fox 1200

## Sensors
Temperature and Humidity: DHT 22
Pressure: BPM 180
Dust: Sharp gp2y1010au0f

## Libraries
DHT Sensor Library
BPM 85
Arduino Low Power
Sigfox MKR
